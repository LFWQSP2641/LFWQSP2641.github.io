Xray 服务端搭建 XTLS + XHTTP 教程

声明：

  • 仅供研究和测试，请勿将其用于任何收费或违法活动，后果自负。
  • 仅供自用，不一定适合其他人。

特点

  1. XTLS 和 XHTTP 可直连
  2. XHTTP 支持过 CDN，并且内层加密，避免中间人 CDN 识别记录
  3. 均使用 tls，避免使用 REALITY 带来的额外负担，如需要筛选合适域名等

准备工作

  1. 域名：需要两个域名，一个设置为服务端 IP，另一个开启 CDN 代理到服务端 IP
  2. 系统：使用 Linux 系系统
